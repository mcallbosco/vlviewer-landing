# Conversations Viewer Landing Page

This is the landing page for the base domain `vlviewer.com`.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## Structure

-   `src/app`: Next.js app directory.
-   `public`: Static assets (icons copied from the main project).
-   `tailwind.config.ts` / `postcss.config.mjs`: Styling configuration.
